# OX 3 จุด

เว็บแอพเกม OX 3x3 สำหรับมือถือ

## กติกา

- ผู้เล่น X และ O วางหมากได้คนละ 3 จุด
- เมื่อทั้งสองฝ่ายวางครบแล้ว จะเข้าสู่ช่วงย้ายหมาก
- เลือกได้ 2 โหมด: บังคับย้ายหมากที่วางเก่าสุด หรือสุ่มหมากที่ต้องย้าย
- ใครเรียงครบ 3 ช่องก่อนชนะ

## ฟีเจอร์

- เล่นได้บนมือถือ
- นับจำนวนเกมทั้งหมด
- นับจำนวนชนะของผู้เล่น X และ O
- เริ่มเกมใหม่ได้โดยไม่ล้างสถิติ
- ล้างสถิติได้เมื่อต้องการ

## วิธีเปิดเล่น

เปิดไฟล์ `index.html` ในเบราว์เซอร์ได้ทันที

## Deploy บน GitHub Pages

1. อัปโหลดไฟล์ทั้งหมดขึ้น GitHub repository
2. ไปที่ `Settings` > `Pages`
3. เลือก `Deploy from a branch`
4. เลือก branch `main` และ folder `/root`
5. กด `Save`

หลังจากนั้น GitHub จะให้ลิงก์สำหรับเปิดเล่นผ่านเว็บ
